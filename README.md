Wordle for C

A simple Wordle clone built using:
- HTML
- CSS
- JavaScript

## Features
- Random word from custom dictionary (words.json)
- All words are related to my relationship
- They are all 6 letter words

## How To Edit Word List

Open words.json and add or remove words.
All words must:
- Be uppercase
- Be same length
- Be inside quotes
- Be separated by commas
